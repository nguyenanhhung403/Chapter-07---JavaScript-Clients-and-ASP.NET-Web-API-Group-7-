CORS Mechanism

Cross-Origin Resource Sharing logic prevents security risks.

01

Preflight Request

Browser automatically sends an OPTIONS request first to check if the server allows the connection.

02

Policy Check

ASP.NET Core Middleware checks Program.cs CORS policy (Origins, Methods, Headers).

03

Access Control

If allowed, server responds with Access-Control-Allow-Origin. Browser then sends the actual data.
